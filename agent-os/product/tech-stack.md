# Tech Stack

### Application Runtime
- **Language:** Python
- **GUI Framework:** Flet (Flutter for Python) for a modern, responsive, and professional UI.

### Git Integration
- **Library:** GitPython (or standard `subprocess` if preferred for simplicity/control).

### AI & API
- **AI Providers:** OpenAI, Anthropic (Claude), Google (Gemini), and Ollama (Local).
- **LLM Orchestration:** LiteLLM (for unified API access across providers) or provider-specific SDKs.

### Configuration
- **Format:** JSON (`config.json`) or Environment Variables (`.env`).
