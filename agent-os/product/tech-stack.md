# Tech Stack

### Application Runtime
- **Language:** Python
- **GUI Framework:** CustomTkinter (or Tkinter) for modern dark-themed UI.

### Git Integration
- **Library:** GitPython (or standard `subprocess` if preferred for simplicity/control).

### AI & API
- **AI Service:** OpenAI API (or Claude equivalent).
- **HTTP Client:** `openai` library or `requests`.

### Configuration
- **Format:** JSON (`config.json`) or Environment Variables (`.env`).
